Notes:

  1. In the table above,
    /test * testClass .. .. ..
    is short for:
    /test plugin = org.cougaar.core.servlet.SimpleServletComponent(testClass, /test) .. .. ..

  2. Many servlets require a "/$agent" prefix to the paths, such as "/$NCA/completion", to specify a specific agent. If a "/$agent" is not specified then a random agent on that node is selected. For some servlets this is fine, such as the "/tasks" servlet that generates an agent-neutral HTML frame. A special-case is the "/agents" servlet, which is a node-level servlet.

  3. Subpaths (e.g. "/test/foo") are fully supported by the server; the examples given above just happen to use simple "/name" paths.
