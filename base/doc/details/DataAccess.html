Cougaar Database Access Information

CSMART and some plugins found in the Cougaar infrastructure and the glm/mlm packages require information from a database. These plugins are able to query a database to gather necessary information using parameters that are set in a file called cougaar.rc or .cougaarrc. You can place a .cougaarrc file in $HOME or you can place a cougaar.rc file in your config directory (e.g. $COUGAAR_INSTALL_PATH/configs/examples/small-135) or the common directory (e.g. $COUGAAR_INSTALL_PATH/configs/common).

The following information provides examples to show the required syntax for these parameters.

The file should define various database drivers that will be used. For example: 
driver.mysql=org.gjt.mm.mysql.Driver
driver.oracle=oracle.jdbc.driver.OracleDriver
driver.postgresql=org.postgresql.Driver

Various Database Connection strings may be used. For example: 
jdbc:mysql://HOST/DBNAME
jdbc:oracle:thin:@HOST:PORT:SID
jdbc:postgresql://HOST:PORT/DBNAME

The file should define parameters used by various plugins. Note that the databases referred to here usually point to at most two distinct databases: 1 for the Cougaar domain data (from 1ADDomainData.sql formerly mysqldump.txt) and one for CSMART configuration data (see the CSMART Installation instructions). Your cougaar.rc file, when running the included societies, would need to define the following information: 

# core database
org.cougaar.database=DBConnectionString
org.cougaar.database.user=User
org.cougaar.database.password=Password

# oplan database
oplan.test.database=DBConnectionString
oplan.test.database.user=User
oplan.test.database.password=Password

# blackjack
blackjack.database=DBConnectionString
blackjack.database.user=User
blackjack.database.password=Password

# blackjack8
blackjack8.database=DBConnectionString
blackjack8.database.user=User
blackjack8.database.password=Password

# icis
icis.database=DBConnectionString
icis.database.user=User
icis.database.password=Password

# generic database expander
generic.database.expander.database=DBConnectionString
generic.database.expander.user=User
generic.database.expander.password=Password
generic.database.expander.expansion_table=construction_expansion

# CSMART configuration database
org.cougaar.configuration.database=DBConnectionString
org.cougaar.configuration.user=User
org.cougaar.configuration.password=Password

# HTTPS keystore and password
# the keystore path is relative to the $COUGAAR_INSTALL_PATH
org.cougaar.web.keystore=Filename
org.cougaar.web.keypass=Password

# datagrabber database configuration
# On the next line, DATAGRABBER_DBNAME is typically "Grabber" but can be
anything you like.
org.cougaar.mlm.ui.grabber.config.DBConfig.connectionURL=jdbc:mysql://HOST/DATAGRABBER_DBNAME
org.cougaar.mlm.ui.grabber.config.DBConfig.user=User  
org.cougaar.mlm.ui.grabber.config.DBConfig.password=Password
org.cougaar.mlm.ui.grabber.config.DBConfig.syntax=MySQL
