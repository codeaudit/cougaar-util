Configuration

Command-line (-D) Properties

org.cougaar.thread.running.max
Provide an integer as the value of this property to set the absolute maximum number of running threads as well as the current max and the size of the thread pool (default = 30). A negative value is interpreted to mean that an unlimited number of threads can run. The current max can be altered dynamically via the ThreadControlService but can never exceed the value given here.

org.cougaar.thread.poolsize.initial
Provide a positive integer as the value of this property to set the number of Java native threads that are instantiated initially in the thread pool.

Servlets

org.cougaar.core.thread.TopPlugin
Dumps the currently running or queued Schedulables. It does not depend on the Metrics Service. This should be loaded into the NodeAgent.

org.cougaar.core.thread.AgentSensorPlugin
Publishes per-agent load average data into the Metrics Service. This should be loaded into the NodeAgent.
