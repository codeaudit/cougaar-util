PSP Services

  • /$NAME/... Refer to a cluster by name
  • ?VERSION
    • displays basic machine information (host, port, OS, ALP-build, JVM, etc)
  • ?LIST
    • Returns an HTML list of all PSPs currently registered
  • ?LOAD?PSP_PATH=Path?CLASSNAME=JName?LOADER=URL
    • Path: logical PSP location at this server (must end with .PSP)
    • JName: Java class name of 'Plan Service Provider'
    • URL: Class is loaded from this URL (optional -- default uses server’s classpath)
  • ?UNLOAD?PSP_PATH=Path?CONFIRM_UNLOAD=Confirm
    • Path: logical PSP location at this server (must end with .PSP)
    • Confirm: Use "Yes", otherwise the user is prompted for a yes/no value
  • ?ECHO
    • reflects browser's incoming HTTP as an HTML document
  • ?FILE?DOCUMENT=PATH
    • it can refer to sub-directories only (no access to parent directories)
  • ?HELP
    • returns this document
