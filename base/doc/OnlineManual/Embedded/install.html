Embedded Cougaar Installation

The Embedded Cougaar install procedure copies files from the Cougaar install path to the new pruned directory. The files needed for the three example configuration are:

Minimum Configuration

  • lib/core.jar
  • lib/util.jar
  • lib/javaiopatch.jar
  • lib/bootstrap.jar
  • lib/mtsstd.jar
  • sys/log4j.jar
  • lib/ping.jar
  • configs/common/NodeAgent.xsl
  • configs/common/SimpleAgent.xsl
  • configs/common/society.xsl
  • configs/common/util.xsl
  • configs/common/make_society.xsl
  • ping/configs/ping/MiniPing-minimum.xml
  • ping/bin/jips.sh
  • ping/bin/test.sh

Standard Configuration

  • Minimum Configuration Plus:
  • lib/mtsstd.jar
  • lib/webtomcat.jar
  • sys/xercesImpl.jar
  • sys/xml-apis.jar
  • sys/servlet.jar
  • sys/tomcat_40.jar
  • webtomcat/data/conf/tomcat-users.xml
  • webtomcat/data/conf/modules.xml
  • webtomcat/data/conf/server.xml
  • webtomcat/data/webapps/ROOT/WEB-INF/web.xml
  • lib/webserver.jar

Metrics Configuration

  • Standard Configuration Plus:
  • lib/qos.jar
  • sys/quoSumo.jar
  • lib/quo.jar
  • ping/configs/ping/MiniPing-Metrics.xml
